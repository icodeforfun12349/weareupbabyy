Antivirus Configuration

Prevent file deletion

‚ñº

‚úì Windows Defender Settings

  1. Open Windows Security (Win + I ‚Üí Privacy & Security)
  2. Go to Virus & threat protection
  3. Turn OFF Real-time protection
  4. Click "Manage settings"
  5. Add exclusion ‚Üí Folder ‚Üí Select Velocity folder
  6. Go to Device Security ‚Üí Core isolation
  7. Turn OFF Memory integrity

‚úì Third-Party Antivirus

  1. Open your antivirus software
  2. Find Exclusions/Whitelist settings
  3. Add entire Velocity folder to exclusions
  4. Add Velocity.exe as trusted application
  5. Disable real-time scanning for gaming
